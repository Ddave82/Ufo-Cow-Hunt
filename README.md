# UFO Cow Hunt

Ein 3D-Browsergame mit Three.js: Fliege nachts mit einem UFO ueber eine Landschaft, finde Kuehe und sammle sie mit dem Lichtstrahl ein.

## Starten

```bash
npm install
npm run dev
```

Dann im Browser oeffnen:

```text
http://127.0.0.1:5173/
```

## Build

```bash
npm run build
```
